The program is written in python language. The output can be generated by running the main.py.
The A* search algorithm is applied. The heuristic used is the number of misplaced tiles. The admissiblity
proof is given in description of main.py. The graphical solution that shows all steps for two of the puzzle
problems requires a screen resolution of at least 1475*768 to fit in the screen. The packages must be installed
if not installed.

Packages:
	pygame
	matplotlib.pyplot
	numpy
	random
	sys
	threading
	time
